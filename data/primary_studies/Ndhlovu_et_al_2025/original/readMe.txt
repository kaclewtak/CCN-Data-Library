Title: 'Dataset: Sediment organic carbon of the Cape Dwarf-Eelgrass Zostera capensis in six South African estuaries'

Authors: Andrew Ndhlovu <andhlovu@sun.ac.za>, Janine B. Adams, Sophie von der Heyden

The data release contains seagrass sediment soil carbon profiles. The data itself is housed in five separate .csv files which can by joined by the core_id and/or site_id attributes. 

To use this data, please cite the following DOI: 10.25573/serc.29847683

Ndhlovu_et_al_2025_materials_and_methods.csv - Contains information on materials and methods broken down by study 
Ndhlovu_et_al_2025_cores.csv - Contains positional and descriptive information on core locations
Ndhlovu_et_al_2025_depthseries.csv - Contains raw depth-series information for all cores
Ndhlovu_et_al_2025_sites.csv - Contains positional information on site locations
Ndhlovu_et_al_2025_species.csv - Contains information on the dominant plant species at coring locations

metadata.xml - Contains a full suite of descriptive metadata with all attribute and variables defined, units clarified, and geographic, temporal, and taxonomic coverages described according to Ecological Metadata Language Standards.

metadata.html - Is a simplified, visual and interactive version of metadata.xml for display purposes.

Ndhlovu_et_al_2025_associated_publication.bib - Contains bibliographic information for publication associated with this data release.
