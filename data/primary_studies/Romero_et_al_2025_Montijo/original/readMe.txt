Dataset: Estimation of blue carbon for the mangrove community of La Playa, Gulf of Montijo, Panama

Authors: Tania E. Romero Gonzalez (romerote@si.edu), Amy Fernandez, Stephanie Vargas, Michel Mojica

Any use of this dataset must include a citation. The DOI: 10.25573/serc.29873261

The data release contains biomass and soil carbon profiles from tidal wetlands. The data itself is housed in separate .csv files which can by joined by relational attributes (ex. site_id and plot_id).

Romero_et_al_2025_sites.csv - Contains site-level information.
Romero_et_al_2025_methods.csv - Contains methodological information on the soil core collection and processing.
Romero_et_al_2025_depthseries.csv - Contains raw depth-series information for all cores.
Romero_et_al_2025_plants.csv - Contains information on the above and belowground plant mass and community composition of each plot.
Romero_et_al_2025_debris.csv - Contains information on the woody debris measured along transects in each plot.

metadata.xml - Contains a full suite of descriptive metadata with all attribute and variables defined, units clarified, and geographic, temporal, and taxonomic coverages described according to Ecological Metadata Language Standards.

metadata.html - Is a simplified, visual and interactive version of metadata.xml for display purposes.
